• Tutorial

    • Getting Started

      A cell is the smallest block in the game. A row, column and region consists of 9 cells and the whole game consists of 81 cells. A region has checkerboard style background color. This simply makes it easier to see where regions are.

      Your objective is to fill a 9x9 grid so that each column, each row, and each of the nine 3x3 boxes (also called blocks or regions) contains the digits from 1 to 9.

      To fix the puzzle to the right, all you have to do is replace the number with the correct green number.
